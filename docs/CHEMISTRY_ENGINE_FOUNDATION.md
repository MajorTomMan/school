# School 通用化学式、反应与化学计量内核

## 元素数据

`PeriodicTable` 保存 118 个元素的稳定身份：原子序数、符号、中文名称、课堂常用相对原子质量和常见氧化态。相对原子质量用于教学计算，不宣称替代带同位素丰度和不确定度的专业数据库。

## 化学式

`ChemicalFormulaParser` 支持：

- 元素符号和下标
- 圆括号与方括号组
- 多层括号
- 结晶水点号和片段系数
- `^2-`、`^+` 等离子电荷
- 固态、液态、气态和水溶液状态

示例：

```text
Al2(SO4)3
CuSO4·5H2O
NH4^+
SO4^2-
Fe^3+(aq)
```

解析后可以得到每种元素的原子数、总电荷、相对式量和各元素质量分数。

## 方程式

`ChemicalEquation` 分开保存反应物、生成物、系数、状态、可逆性和反应条件。守恒检查同时比较：

- 每种元素的原子总数
- 方程式两侧总电荷

`ChemicalEquationBalancer` 把元素和电荷写成有理数矩阵，求齐次线性方程组的零空间，再转换为最小正整数系数。结果必须再次通过守恒复核。

它可以处理普通分子方程式和给定物种的离子方程式，但不会仅凭反应物猜测未知产物。产物和反应条件仍由教材题目或明确的反应规则提供。

## 离子方程式

`IonicEquationReducer` 可以在已完整写出的离子方程式两侧约去状态、组成和电荷均相同的旁观离子。它不负责决定哪些弱电解质应拆分；拆分规则属于具体教材章节和反应规则。

## 化学计量

`Stoichiometry` 提供：

- 相对分子质量和摩尔质量
- 质量与物质的量换算
- 物质的量浓度
- 稀释
- 反应进度
- 限量试剂
- 反应物剩余量
- 生成物理论量
- 理论产率与实际产率百分比

计算以已经配平的方程式为前提。单位展示和有效数字由共享科学内核负责。

## 边界

本轮没有宣称完成：

- 任意反应产物预测
- 水溶液物种自动拆分
- 氧化数自动分配的所有例外
- 半反应法自动配平的完整酸碱介质规则
- 活度、非理想溶液和复杂平衡
- 反应动力学和热力学数据库

这些能力可以在高中具体教材要求出现时继续扩展。当前内核已经能够可靠支撑化学式、质量守恒、电荷守恒、常见方程式配平和基础化学计量。
