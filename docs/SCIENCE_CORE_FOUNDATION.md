# School 数学、物理与化学科学内核

## 目标

科学内核用于承载数学、物理和化学课程共同需要的确定性能力。教材课程仍决定知识顺序、允许概念和题目语境；内核只提供可复用的数据结构、计算、诊断、步骤和可视化接口。

本轮按五个串联 PR 建设：

1. 精确表达式、根号、π、单位、量纲和有效数字。
2. 多项式、方程、不等式、函数、向量、几何和证明步骤。
3. 物理模型、直流电路、拓扑诊断、电功率和能量。
4. 元素、化学式、反应方程式、矩阵配平、离子和化学计量。
5. 有机分子图、结构式、官能团、异构体和教材限定反应变换。

## 精确值

`BigRational` 使用任意精度整数保存分子和分母。有限小数通过十进制位数直接转换，不经过 `Double`：

```text
0.1  -> 1/10
1/3 + 1/6 -> 1/2
```

`ScienceExpression` 是表达式 AST，V1 支持：

- 整数、有限小数和分数
- π、e、i
- 变量
- 加减乘除
- 隐式乘法
- 整数幂
- 平方根
- 实数域与复数域边界

基础化简保留精确形式，例如：

```text
√72 -> 6√2
2π + 3π -> 5π
√(-8) -> 实数域拒绝；复数域为 2i√2
```

近似值只在显示或数值模拟需要时由 `ScienceExpressionEvaluator` 生成。

## 物理量、单位与量纲

`Dimension` 使用七个 SI 基本量纲指数：

```text
长度 L
质量 M
时间 T
电流 I
温度 Θ
物质的量 N
发光强度 J
```

导出量纲由乘、除和幂组合产生。单位换算以 SI 为中间值并保持有理数精度，摄氏温度使用仿射偏移。

首批单位包括长度、面积、体积、时间、质量、电流、温度、物质的量、速度、加速度、力、压强、能量、功率、电荷、电压、电阻、电容、频率和密度常用单位。

示例：

```text
72 km/h -> 20 m/s
1 kWh -> 3 600 000 J
20 °C -> 293.15 K
```

不同量纲不能相加或换算。后续物理公式和化学计算必须使用这一层，不允许只比较单位字符串。

## 测量

`MeasuredValue` 和 `SignificantFigures` 为实验课程提供：

- 有效数字计数
- 指定位数舍入
- 绝对不确定度
- 相对不确定度
- 百分比不确定度

复杂误差传播会在教材真正使用时继续扩展。

## 权限边界

底层支持某个符号或操作，不代表所有课程都能使用。课程必须通过 `LessonCapability` 显式允许：

- 精确数、根式和 π
- 表达式解析、化简和近似
- 物理量、单位和量纲
- 单位换算和有效数字
- 对应实验室组件

例如初中实数课程可以允许平方根但不允许虚数单位；复数课程才允许负数平方根进入复数域。

## 性能与安全边界

- 表达式文本限制长度和 token 数量。
- 指数首版只接受整数。
- 根式首版重点处理平方根和教材常用整数根式。
- 不执行任意代码，不调用脚本解释器。
- 不承诺 Mathematica 级通用符号化简。
- 所有规则必须有确定性单元测试。

## 实验室

路径：

```text
实验室 -> 科学内核
```

V1 包含精确表达式和单位量纲两个样板，后续四个 PR 会在同一入口增加数学、物理电路、通用化学和有机化学验证页。
