## 修改点

- APK 改为纯课程运行时，不再打包教材目录、章节标题、课程正文、例题、小结、习题或学科课程文案。
- 课程目录和学习页面全部从已安装的云端课程包生成；课程包负责提供教材、章节、小节、页码和页面块数据。
- 应用每次冷启动在首帧之后后台检查课程清单，不阻塞首页显示。
- 每本教材同时支持完整 ZIP 和独立课程文件；首次安装、结构升级或大范围修改使用全量下载，小范围修改使用文件级增量下载。
- 增量文件总大小低于完整包的 60% 时选择增量更新，否则自动下载完整课程包；增量安装失败时自动回退全量包。
- 课程下载先进入 `staging`，完成文件大小、SHA-256、安全路径和 JSON 结构校验后再切换到 `active`；旧版本保留在 `backup`。
- APK 继续保留课程数据结构、解析器、缓存与回滚、横向翻页渲染器、可视化组件以及数学、物理、化学、生物等确定性算法。
- 新增课程数据缺失页面；没有缓存或同步失败时显示状态和重新同步入口，不再生成或回退任何内置课程。

## 修复点

- 删除旧的有理数硬编码课程、各学科通用课程工厂、课程专用工作台和预制课程分析文案。
- 删除 APK assets 中的数学知识索引与全学科预制教材目录，减少安装包内的课程数据。
- 覆盖安装时自动清理旧版本写入的未绑定 `prebuilt-*` 课程，同时保留用户主动导入并绑定的教材 PDF。
- 云端课程缓存会映射为本地教材与章节导航；课程更新后无需重新发布 APK。
- 课程包不能下发 Kotlin、JavaScript 或其他可执行代码，只能选择 APK 已注册的数据块、可视化和验证能力。
