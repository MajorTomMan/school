## 修改点

- 新增云端课程包体系：APK 保留数据结构、页面渲染器、可视化和确定性算法，课程正文可由远端 JSON 更新。
- 应用每次冷启动在首帧之后后台检查课程清单，不阻塞首页显示。
- 每本教材同时支持完整 ZIP 和独立课程文件；首次安装、结构升级或大范围修改使用全量下载，小范围修改使用文件级增量下载。
- 增量文件总大小低于完整包的 60% 时选择增量更新，否则自动下载完整课程包；增量安装失败时自动回退全量包。
- 课程下载先进入 `staging`，完成文件大小、SHA-256、安全路径和 JSON 结构校验后再切换到 `active`；旧版本保留在 `backup`。
- 云端不可访问、校验失败或课程包损坏时继续使用上一次有效缓存；没有缓存时继续使用 APK 内置课程。
- 云端课程按照章节、小节、学习页和页面块组织，支持教材正文、公式、例题、小结、练习、历史材料和已注册可视化。
- 新增 Google Drive 分享链接转换，构建时可通过 `SCHOOL_COURSE_MANIFEST_URL` 或 `schoolCourseManifestUrl` 配置课程清单。
- 新增课程发布工具，可根据 `course.json` 生成独立增量文件、完整 ZIP、SHA-256 与远端清单。

## 修复点

- “有理数的大小比较”等教材标题会直接进入有理数横向翻页课程，不再落入旧的上下滚动通用数学页面。
- 云端缓存中存在匹配课程时优先使用云端页面；缓存不存在或同步失败时使用现有内置翻页课程。
- 课程包不允许通过云端下发 Kotlin、JavaScript 或其他可执行代码，只能选择 APK 已注册的页面块和可视化类型。
- 新增课程清单解析、标题别名、章节复习、练习页、Google Drive 链接、增量选择、全量回退和删除文件测试。
