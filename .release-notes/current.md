## 修改点

- 设置页重构为“代理”和“AI”两个页面，移除原有学科管理入口。
- 新增应用级代理配置，支持 HTTP、HTTPS、SOCKS 和 SOCKS5 地址。
- 代理用途拆分为“版本更新走代理”和“AI 请求走代理”两个独立开关。
- 应用更新页移入代理页面，继续保留自动检查、仅 Wi-Fi 下载、手动检查和状态查看。
- AI 页面保留接口地址、模型、API Key、连接测试和本地学习记录清理功能。

## 修复点

- 更新清单、签名文件和 APK 下载现在会按代理设置建立连接，不再只能直接访问 GitHub 发布通道。
- AI 请求可选择复用同一代理，也可以继续直连，不会被更新代理开关连带影响。
- 代理设置只作用于 School 自身指定的网络请求，不修改 Android 系统全局代理。
- 新增代理地址解析和按用途路由测试，覆盖 HTTP 默认协议、SOCKS5、直连和无效地址。
